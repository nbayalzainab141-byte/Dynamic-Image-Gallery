# Dynamic Image Gallery

A simple and responsive Dynamic Image Gallery built using HTML, CSS, and Vanilla JavaScript. This project focuses on fundamental DOM manipulation and dynamic rendering concepts.

---

## Features

- Dynamic image rendering using JavaScript
- Structured image data using arrays and objects
- Clean and responsive card-based UI
- Simple and beginner-friendly layout
- Lightweight and fast performance

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## About Project

This project demonstrates how to create a dynamic UI using JavaScript without any frameworks. It helps in understanding how DOM manipulation and arrays work together.
